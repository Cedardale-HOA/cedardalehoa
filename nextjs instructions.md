# Next JS Proof of Concept Instructions

## Purpose
Using the design of index-v2.html to create a nextjs site that uses Sanity Headless CMS as the content source of truth.

## Expectations

- Non technical users should be able to create events to be listed on the website.
    - The next event should be on the front page and a dedicated page should have a list of events
- I would also like to have a page of blog articles and a page that lists the current members and officers of the board.
- As much content as possible should be stored and used via the Sanity.io CMS system and organized in an easy to understand and edit way for non-technical users
- Use the Using Sanity link in references to teach yourself how to get started with my sanity project.
